# AviClip
Портативная программа расширения методов работы с системным текстовым буфером обмена. ПО не требует установки дополнительных библиотек и полностью бесплатно! Для использования достаточно принять пользовательское соглашение. Работает на Windows 7 и выше.
Программа изготовлена в рамках проектной деятельности и не является коммерческим продуктом. В связи с этим, автор не несёт ответственности за её работу. AviClip изготовлена в Embarcadero RAD Studio 12 C++ и сжата через UPX.
